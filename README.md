# NES Emulator

An eventual cycle accurate NES emulator

## Accuracy

* All of the CPU works - Both official and unofficial opcodes
* PPU can play most games
* APU mostly works, except for DMC

## Mappers Supported

* NROM
* MMC1
* UxROM
* MMC3 - Mostly Working

## Details

* Built with C++
* GLFW for window and input
* SDL for audio
